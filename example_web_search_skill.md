# 网页搜索与分析技能

提供强大的网络搜索和内容分析功能。

## 功能特性

### 多引擎搜索
- Brave 搜索引擎
- DuckDuckGo 搜索引擎
- 自动去重和排序

### 深度研究
- 多源信息收集
- 内容交叉验证
- 综合分析报告

### 竞品分析
- 竞品信息收集
- 功能对比分析
- 市场趋势分析

## 使用方法

### 网络搜索
用户说："搜索人工智能的最新发展"
AI 响应："我为你找到了以下关于人工智能最新发展的信息..."

### 深度调研
用户说："帮我调研一下竞品"
AI 响应："我正在为你调研竞品信息..."

### 内容分析
用户说："分析这个网页的内容"
AI 响应："这个网页的主要内容是..."
